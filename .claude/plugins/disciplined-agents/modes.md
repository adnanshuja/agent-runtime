# Disciplined Agents — Modes (Plugin Module)

Reference: `modes/`

STRICT (default): Full 6 stages. R1-R8 all strict. Scorecard FAIL on any<6. Confirm S3+S5.
FAST (LOW only): Abbreviated. R1/R2/R4/R7 enforced. Scorecard optional. No pauses.
DEBUG (investigation): S0 reproduce. Hypothesis tree. R0 root cause required. R1-R8 same as STRICT.
